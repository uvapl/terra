# Code layout

`root`
: main apps

`commands`
: command layer for menus and toolbars

`components`
: user interface components with internal operations

`concerns`
: various mixins for the main apps (filetree, storage, git, lfs)

`controllers`
: view controllers for each app

`fs`
: various file system implementations

`layouts`
: general user interface and layout per app

`lib`
: utilities and helpers

`vendor`
: libraries

`workers`
: programming language runtimes in WASM
